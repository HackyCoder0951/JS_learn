# JavaScript Operators

## Unary Operator

An operator is **unary** if it has a single operand.

### Example:
```js
let x = 1;
x = -x;

alert(x); // -1
```

- Unary negation `-` reverses the sign of a number.

### More Examples:
```js
let a = 5;

console.log(-a); // -5
console.log(+a); // 5
console.log(typeof a); // number
```

---

## Binary Operator

An operator is **binary** if it has two operands.

### Example:
```js
let x = 1, y = 3;

alert(y - x); // 2
```

- Binary minus `-` subtracts values.

### More Examples:
```js
console.log(10 + 5); // 15
console.log(20 - 10); // 10
console.log(4 * 5); // 20
console.log(20 / 4); // 5
```

---

## Operand

An operand is what operators are applied to.

### Example:
```js
5 * 2
```

- Left operand = `5`
- Right operand = `2`

### More Examples:
```js
10 + 20
// 10 and 20 are operands

x++
/*
x is operand
++ is operator
*/
```

## General Notes

### 01 Not all syntax included here are "operators" in the strict sense.

For example:
- Spread `...`
- Arrow `=>`

These are included to show how tightly they bind compared to other expressions.

### 02 Some operators require narrower expressions.

Example:
```js
obj.property
```

- Right-hand side of `.` must be an identifier.

### 03 Some operators accept wider expressions.

Example:
```js
arr[(1, 2, 3)]
```

- Inside `[]`, comma expressions are allowed.

# Short-Circuiting

Short-circuiting means conditional evaluation.

If the first condition already determines the result, the second expression is not evaluated.

## Examples

```js
a || (b * c);
```
- If `a` is truthy, `(b * c)` is not evaluated.

```js
a && (b < c);
```
- If `a` is falsy, `(b < c)` is not evaluated.

```js
a ?? (b || c);
```
- If `a` is not `null` or `undefined`, right side is skipped.

```js
a?.b.c;
```
- If `a` is `null` or `undefined`, result becomes `undefined`.

---

## More Examples

```js
let username = "Admin";

console.log(username || "Guest");
// Admin
```

```js
let isLoggedIn = false;

console.log(isLoggedIn && "Welcome");
// false
```

```js
let data = null;

console.log(data ?? "Default Value");
// Default Value
```

# JavaScript Arithmetic Operators

Arithmetic operators perform mathematical operations.

| Operator | Description | Example |
|---|---|---|
| `+` | Addition | `10 + 20 = 30` |
| `-` | Subtraction | `20 - 10 = 10` |
| `*` | Multiplication | `10 * 20 = 200` |
| `/` | Division | `20 / 10 = 2` |
| `%` | Modulus | `20 % 10 = 0` |
| `++` | Increment | `a++` |
| `--` | Decrement | `a--` |
| `**` | Exponentiation | `10 ** 2 = 100` |
| `**=` | Exponentiation Assignment | `x **= 2` |

---

## Examples of JS Arithmetic Operations

```js
console.log(10 + 20);
console.log(10 - 20);
console.log(10 * 20);
console.log(10 / 20);
console.log(10 % 20);
```

### Increment
```js
let a = 10;
a++;

console.log(a); // 11
```

### Decrement
```js
let a = 10;
a--;

console.log(a); // 9
```

### Exponentiation
```js
let x = 10;

console.log(x ** 2); // 100
```

### Exponentiation Assignment
```js
let x = 10;

x **= 2;

console.log(x); // 100
```

### Additional Examples

```js
console.log(5 + 5); // 10
console.log(15 % 4); // 3
console.log(2 ** 5); // 32
```

# JavaScript Comparison Operators

Comparison operators compare two operands.

| Operator | Description | Example |
|---|---|---|
| `==` | Equal to | `10 == 20` |
| `===` | Strict equal | `10 === 20` |
| `!=` | Not equal | `10 != 20` |
| `!==` | Strict not equal | `20 !== 20` |
| `>` | Greater than | `20 > 10` |
| `>=` | Greater than or equal | `20 >= 10` |
| `<` | Less than | `20 < 10` |
| `<=` | Less than or equal | `20 <= 10` |

---

## Examples

```js
console.log(10 == "10"); // true
console.log(10 === "10"); // false
console.log(20 > 10); // true
console.log(20 <= 10); // false
```

### Additional Examples

```js
console.log(5 != 10); // true
console.log(15 >= 15); // true
console.log(8 < 20); // true
```

---

# JavaScript Bitwise Operators

Bitwise operators work on binary numbers.

| Operator | Description |
|---|---|
| `&` | Bitwise AND |
| `|` | Bitwise OR |
| `^` | Bitwise XOR |
| `~` | Bitwise NOT |
| `<<` | Left Shift |
| `>>` | Right Shift |
| `>>>` | Unsigned Right Shift |

---

## Examples

```js
console.log(5 & 1); // 1
console.log(5 | 1); // 5
console.log(5 ^ 1); // 4
console.log(~5); // -6
console.log(5 << 1); // 10
console.log(5 >> 1); // 2
```

### Additional Examples

```js
console.log(8 << 2); // 32
console.log(16 >> 2); // 4
```

---

# JavaScript Logical Operators

| Operator | Description |
|---|---|
| `&&` | Logical AND |
| `||` | Logical OR |
| `!` | Logical NOT |

---

## Examples

```js
console.log(true && false); // false
console.log(true || false); // true
console.log(!true); // false
```

### Additional Examples

```js
console.log(10 > 5 && 20 > 10); // true
console.log(10 < 5 || 20 > 10); // true
console.log(!(10 > 5)); // false
```

## Yield Operator

```js
function* generator() {
    const value = yield 1;
    console.log(value);
}

const iter = generator();

iter.next();
iter.next("hello");
```

### Additional Example

```js

    yield 10;
    yield 20;
}

const num = numbers();

console.log(num.next());
console.log(num.next());
```
