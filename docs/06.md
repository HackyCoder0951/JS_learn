# JavaScript `if-else` Statement

The JavaScript `if-else` statement is used to execute code based on conditions.

There are three forms of conditional statements in JavaScript:

1. `if` Statement
2. `if...else` Statement
3. `if...else if...else` Statement

---

# 01 - `if` Statement

## Description
The `if` statement executes a block of code only when the condition is `true`.

---

## Syntax

```javascript
if (condition) {
    // code to execute if condition is true
}
```

---

## Example

```javascript
{
    let a = 10;
    let b = 24;

    if (a < b) {
        console.log("Condition is true: a = 10 < b = 24");
    }
}
```

---

## Output

```javascript
Condition is true: a = 10 < b = 24
```

---

## Another Example

```javascript
{
    let age = 18;

    if (age >= 18) {
        console.log("You are eligible to vote.");
    }
}
```

---

## Output

```javascript
You are eligible to vote.
```

---

# 02 - `if...else` Statement

## Description
The `if...else` statement executes one block of code when the condition is `true`, otherwise it executes another block of code.

---

## Syntax

```javascript
if (condition) {
    // code executes if condition is true
} else {
    // code executes if condition is false
}
```

---

## Example

```javascript
{
    let a = 10;
    let b = 24;

    if (a > b) {
        console.log("Condition is true: a > b");
    } else {
        console.log("Condition is false: a < b");
    }
}
```

---

## Output

```javascript
Condition is false: a < b
```

---

## Another Example

```javascript
{
    let number = 7;

    if (number % 2 == 0) {
        console.log("Even Number");
    } else {
        console.log("Odd Number");
    }
}
```

---

## Output

```javascript
Odd Number
```

---

# 03 - `if...else if...else` Statement

## Description
The `if...else if...else` statement is used to test multiple conditions.

JavaScript checks conditions from top to bottom and executes the first `true` condition.

---

## Syntax

```javascript
if (condition1) {
    // executes if condition1 is true
} else if (condition2) {
    // executes if condition2 is true
} else if (condition3) {
    // executes if condition3 is true
} else {
    // executes if all conditions are false
}
```

---

## Example

```javascript
{
    let a = 10;
    let b = 24;

    if (a < b) {
        console.log("Condition1 is true: a < b");
    } else if (a > b) {
        console.log("Condition2 is true: a > b");
    } else if (a == b) {
        console.log("Condition3 is true: a == b");
    } else {
        console.log("No condition is true");
    }
}
```

---

## Output

```javascript
Condition1 is true: a < b
```

---

## Another Example

```javascript
{
    let marks = 75;

    if (marks >= 90) {
        console.log("Grade A");
    } else if (marks >= 70) {
        console.log("Grade B");
    } else if (marks >= 50) {
        console.log("Grade C");
    } else {
        console.log("Fail");
    }
}
```

---

## Output

```javascript
Grade B
```

# Important Notes

## `=` vs `==` vs `===`

| Operator | Meaning | Example |
|---|---|---|
| `=` | Assignment Operator | `a = 10` |
| `==` | Equality Operator | `a == b` |
| `===` | Strict Equality Operator | `a === b` |

---

## Example

```javascript
{
    let a = 10;
    let b = "10";

    console.log(a == b);   // true
    console.log(a === b);  // false
}
```

---

## Output

```javascript
true
false
```

# Real-World Example

## Login System Example

```javascript
{
    let username = "admin";
    let password = "12345";

    if (username === "admin" && password === "12345") {
        console.log("Login Successful");
    } else {
        console.log("Invalid Username or Password");
    }
}
```

---

## Output

```javascript
Login Successful
```

# Summary

| Statement | Purpose |
|---|---|
| `if` | Executes code only when condition is true |
| `if...else` | Executes one block if true and another if false |
| `if...else if...else` | Checks multiple conditions |
