# Looping in JavaScript

---

# Introduction to Loops

## What is a Loop?

- Loops are used to execute a block of code repeatedly.
- Loops help reduce:
  - Repetitive code
  - Manual work
  - Duplicate statements

---

# Without Loop

```js
console.log("Hello");
console.log("Hello");
console.log("Hello");
console.log("Hello");
console.log("Hello");
```

---

# With Loop

```js
for (let i = 1; i <= 5; i++) {

    console.log("Hello");
}
```

---

# Why Loops are Important?

- Data iteration
- Array traversal
- API data handling
- DOM manipulation
- Automation
- Backend processing

---

# Types of Loops in JavaScript

| Loop | Purpose |
|---|---|
| `for` | Fixed number of iterations |
| `while` | Runs while condition is true |
| `do...while` | Executes at least once |
| `for...of` | Iterates values |
| `for...in` | Iterates object keys |

---

# 01 - `for` Loop

## Description

- The `for` loop runs code a fixed number of times.

## Syntax

```js
for (initialization; condition; increment/decrement) {

    // code
}
```

---

# Flow of `for` Loop

```text
1. Initialization
2. Condition Check
3. Execute Code
4. Increment/Decrement
5. Repeat
```

---

# Example

```js
{
    for (let i = 1; i <= 5; i++) {

        console.log(
            "01 - for Loop = ",
            i
        );
    }
}
```

---

# Reverse Loop

## Example

```js
{
    for (let i = 5; i >= 1; i--) {

        console.log(i);
    }
}
```

---

# Infinite Loop

## Warning

- Infinite loops never stop.
- They can crash browser or application.

## Example

```js
// Dangerous

for (;;) {

    console.log("Infinite Loop");
}
```

---

# 02 - `while` Loop

## Description

- The `while` loop runs while a condition remains true.

## Syntax

```js
while (condition) {

    // code
}
```

---

# Example

```js
{
    let i = 1;

    while (i <= 5) {

        console.log(
            "02 - while Loop = ",
            i
        );

        i++;
    }
}
```

---

# Real World Example - Password Attempts

```js
let attempts = 1;

while (attempts <= 3) {

    console.log(
        `Attempt ${attempts}`
    );

    attempts++;
}
```

---

# 03 - `do...while` Loop

## Description

- The `do...while` loop executes code at least once.
- Condition is checked after execution.

## Syntax

```js
do {

    // code

} while (condition);
```

---

# Example

```js
{
    let i = 1;

    do {

        console.log(
            "03 - do while Loop = ",
            i
        );

        i++;

    } while (i <= 5);
}
```

---

# Difference Between `while` and `do...while`

| `while` | `do...while` |
|---|---|
| Checks condition first | Executes once before checking |
| May not run | Runs at least once |

---

# Example

## `while`

```js
{
    let i = 10;

    while (i <= 5) {

        console.log(i);
    }
}
```

### Output

```js
No Output
```

---

## `do...while`

```js
{
    let i = 10;

    do {

        console.log(i);

    } while (i <= 5);
}
```

### Output

```js
10
```

# 04 - `for...of` Loop

## Description

- The `for...of` loop iterates over iterable values.
- Mainly used with:
  - Arrays
  - Strings
  - Maps
  - Sets

## Syntax

```js
for (variable of iterable) {

    // code
}
```

---

# Example with Array

```js
{
    let fruits = [
        "Apple",
        "Banana",
        "Mango"
    ];

    for (let fruit of fruits) {

        console.log(
            "04 - for...of Loop = ",
            fruit
        );
    }
}
```

---

# Example with String

```js
{
    let name = "Jignesh";

    for (let char of name) {

        console.log(char);
    }
}
```

---

# 05 - `for...in` Loop

## Description

- The `for...in` loop iterates over object keys.

## Syntax

```js
for (key in object) {

    // code
}
```

---

# Example

```js
{
    let user = {
        name: "Jignesh",
        age: 21,
        city: "Delhi"
    };

    for (let key in user) {

        console.log(
            key,
            user[key]
        );
    }
}
```

---

# Difference Between `for...of` and `for...in`

| `for...of` | `for...in` |
|---|---|
| Iterates values | Iterates keys |
| Used for arrays | Used for objects |

---

# Nested Loops

## Description

- A loop inside another loop is called nested loop.

## Example

```js
{
    for (let i = 1; i <= 3; i++) {

        for (let j = 1; j <= 2; j++) {

            console.log(i, j);
        }
    }
}
```

---

# Pattern Example

```js
{
    for (let i = 1; i <= 5; i++) {

        let pattern = "";

        for (let j = 1; j <= i; j++) {

            pattern += "*";
        }

        console.log(pattern);
    }
}
```

### Output

```js
*
**
***
****
*****
```

---

# `break` Statement

## Description

- The `break` statement stops the loop completely.

## Example

```js
{
    for (let i = 1; i <= 10; i++) {

        if (i === 5) {

            break;
        }

        console.log(i);
    }
}
```

---

# `continue` Statement

## Description

- The `continue` statement skips current iteration.

## Example

```js
{
    for (let i = 1; i <= 5; i++) {

        if (i === 3) {

            continue;
        }

        console.log(i);
    }
}
```

---

# Real World Example - Search User

```js
let users = [
    "Rahul",
    "Aman",
    "Jignesh"
];

for (let user of users) {

    if (user === "Jignesh") {

        console.log("User Found");

        break;
    }
}
```

---

# Real World Example - Even Numbers

```js
for (let i = 1; i <= 10; i++) {

    if (i % 2 !== 0) {

        continue;
    }

    console.log(i);
}
```

---

# Important Notes

- Loops are fundamental in programming.
- Avoid unnecessary nested loops.
- Infinite loops can freeze applications.
- Use proper loop type based on use case.
