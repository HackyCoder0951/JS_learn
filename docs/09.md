# JavaScript Array Methods

---

# Introduction

- JavaScript arrays are used to store multiple values in a single variable.
- Arrays can store:
  - Numbers
  - Strings
  - Objects
  - Functions
  - Other arrays
  - Mixed datatypes

---

# Why Arrays are Used?

## Without Array

```js
let student1 = "Jignesh";
let student2 = "Rahul";
let student3 = "Aman";
```

- Managing multiple variables becomes difficult.

---

## With Array

```js
let students = ["Jignesh", "Rahul", "Aman"];
```

- Easier to manage data.
- Better for looping and operations.

---

# Creating Arrays

## 1) Using Array Literal (Recommended)

## Syntax

```js
let arrayName = [value1, value2, value3];
```

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits);
```

---

# 2) Using `new Array()`

## Syntax

```js
let arrayName = new Array(value1, value2);
```

## Example

```js
let fruits = new Array("Apple", "Banana", "Mango");

console.log(fruits);
```

---

# Array Can Store Mixed Data Types

## Example

```js
let mixed = [
    "Jignesh",
    21,
    true,
    null,
    undefined
];

console.log(mixed);
```

---

# Accessing Array Elements

- Array indexing starts from `0`.

## Syntax

```js
arrayName[index]
```

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits[0]);

console.log(fruits[1]);

console.log(fruits[2]);
```

---

# Updating Array Elements

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

fruits[1] = "Orange";

console.log(fruits);
```

---

# Array Length Property

## Description

- The `length` property returns total number of elements in an array.

## Syntax

```js
array.length
```

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits.length);
```

---

# Last Element of Array

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

console.log(
    fruits[fruits.length - 1]
);
```

---

# Nested Arrays

## Description

- Arrays inside another array are called nested arrays.

## Example

```js
let nested = [
    ["HTML", "CSS"],
    ["JavaScript", "React"]
];

console.log(nested[0]);

console.log(nested[1][1]);
```

---

# Array Destructuring

## Description

- Destructuring extracts values from arrays into variables.

## Example

```js
let colors = ["Red", "Green", "Blue"];

let [a, b, c] = colors;

console.log(a);

console.log(b);

console.log(c);
```

---

# Array Using Loop

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

for (let i = 0; i < fruits.length; i++) {

    console.log(fruits[i]);
}
```

# Array Methods

---

# 01 - Array `push()` Method

## Description

- The `push()` method adds elements to the end of an array.
- It modifies the original array.

## Syntax

```js
array.push(element1, element2)
```

## Example

```js
{
    let fruits = ["Apple", "Banana"];

    fruits.push("Mango");

    console.log(
        "01 - Array 'push()' Method = ",
        fruits
    );
}
```

---

# 02 - Array `pop()` Method

## Description

- The `pop()` method removes the last element from an array.
- It returns the removed element.

## Syntax

```js
array.pop()
```

## Example

```js
{
    let fruits = ["Apple", "Banana", "Mango"];

    let removed = fruits.pop();

    console.log(
        "02 - Array 'pop()' Method = ",
        removed
    );

    console.log(fruits);
}
```

---

# 03 - Array `shift()` Method

## Description

- The `shift()` method removes the first element from an array.

## Syntax

```js
array.shift()
```

## Example

```js
{
    let fruits = ["Apple", "Banana", "Mango"];

    let removed = fruits.shift();

    console.log(
        "03 - Array 'shift()' Method = ",
        removed
    );

    console.log(fruits);
}
```

---

# 04 - Array `unshift()` Method

## Description

- The `unshift()` method adds elements at the beginning of an array.

## Syntax

```js
array.unshift(element1, element2)
```

## Example

```js
{
    let fruits = ["Banana", "Mango"];

    fruits.unshift("Apple");

    console.log(
        "04 - Array 'unshift()' Method = ",
        fruits
    );
}
```

---

# Difference Between `push()` and `unshift()`

| `push()` | `unshift()` |
|---|---|
| Adds element at end | Adds element at beginning |
| Faster | Slower |

---

# Difference Between `pop()` and `shift()`

| `pop()` | `shift()` |
|---|---|
| Removes from end | Removes from beginning |
| Faster | Slower |

# Important Notes

- Arrays are mutable.
- Arrays are objects in JavaScript.
- Arrays use zero-based indexing.
- Arrays can grow dynamically.
- Array methods can:
  - Modify original array
  - Return new array
  - Return values
