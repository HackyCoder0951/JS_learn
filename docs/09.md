# JavaScript Array Methods

---

# Introduction

- JavaScript arrays are used to store multiple values in a single variable.
- Arrays can store:
  - Numbers
  - Strings
  - Objects
  - Functions
  - Other arrays
  - Mixed datatypes

---

# Why Arrays are Used?

## Without Array

```js
let student1 = "Jignesh";
let student2 = "Rahul";
let student3 = "Aman";
```

- Managing multiple variables becomes difficult.

---

## With Array

```js
let students = ["Jignesh", "Rahul", "Aman"];
```

- Easier to manage data.
- Better for looping and operations.

---

# Creating Arrays

## 1) Using Array Literal (Recommended)

## Syntax

```js

```

## Example

```js
let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits);
```

---

# 2) Using `new Array()`

## Syntax

```js

```

## Example


# Array Methods

---

# 01 - Array `push()` Method

## Description

- The `push()` method adds elements to the end of an array.
- It modifies the original array.

## Syntax

```js
array.push(element1, element2)
```

## Example

```js
{
    let fruits = ["Apple", "Banana"];

    fruits.push("Mango");

    console.log(
        "01 - Array 'push()' Method = ",
        fruits
    );
}
```

---

# 02 - Array `pop()` Method

## Description

- The `pop()` method removes the last element from an array.
- It returns the removed element.

## Syntax

```js
array.pop()
```

## Example

```js
{
    let fruits = ["Apple", "Banana", "Mango"];

    let removed = fruits.pop();

    console.log(
        "02 - Array 'pop()' Method = ",
        removed
    );

    console.log(fruits);
}
```

---

# 03 - Array `shift()` Method

## Description

- The `shift()` method removes the first element from an array.

## Syntax

```js
array.shift()
```

## Example

```js
{
    let fruits = ["Apple", "Banana", "Mango"];

    let removed = fruits.shift();

    console.log(
        "03 - Array 'shift()' Method = ",
        removed
    );

    console.log(fruits);
}
```

---

# 04 - Array `unshift()` Method

## Description

- The `unshift()` method adds elements at the beginning of an array.

## Syntax

```js
array.unshift(element1, element2)
```

## Example

```js
{
    let fruits = ["Banana", "Mango"];

    fruits.unshift("Apple");

    console.log(
        "04 - Array 'unshift()' Method = ",
        fruits
    );
}
```

---

# Difference Between `push()` and `unshift()`

| `push()` | `unshift()` |
|---|---|
| Adds element at end | Adds element at beginning |
| Faster | Slower |

---

# Difference Between `pop()` and `shift()`

| `pop()` | `shift()` |
|---|---|
| Removes from end | Removes from beginning |
| Faster | Slower |

# 05 - Array `concat()` Method

## Description

- The `concat()` method merges two or more arrays.
- It returns a new array.
- It does not modify the original arrays.

## Syntax

```js
array1.concat(array2, array3)
```

## Example

```js

    let fullStack = frontend.concat(backend);

    console.log(
        "05 - Array 'concat()' Method = ",
        fullStack
    );
}
```

# 08 - Array `splice()` Method

## Description

- The `splice()` method adds or removes elements from an array.
- It modifies the original array.

## Syntax

```js
array.splice(start, deleteCount, item1, item2)
```

## Example - Remove Elements

```js
{
    let fruits = [
        "Apple",
        "Banana",
        "Mango"
    ];

    fruits.splice(1, 1);

    console.log(
        "08 - Array 'splice()' Remove Method = ",
        fruits
    );
}
```

---

## Example - Add Elements

```js
{
    let fruits = [
        "Apple",
        "Mango"
    ];

    fruits.splice(1, 0, "Banana");

    console.log(
        "08.1 - Array 'splice()' Add Method = ",
        fruits
    );
}
```

---

# Difference Between `slice()` and `splice()`

| `slice()` | `splice()` |
|---|---|
| Does not modify original array | Modifies original array |
| Returns extracted part | Adds/removes elements |
| Safer | More powerful |

# Important Note About Number Sorting

## Wrong Way

```js
{
    let numbers = [100, 25, 3, 8];

    numbers.sort();

    console.log(numbers);
}
```

### Output

```js
[100, 25, 3, 8]
```

---

## Correct Way

```js
{
    let numbers = [100, 25, 3, 8];

    numbers.sort((a, b) => a - b);

    console.log(numbers);
}
```

### Output

```js
[3, 8, 25, 100]
```

---

# Ascending Order Sorting

```js
numbers.sort((a, b) => a - b);
```

---

# Descending Order Sorting

```js
numbers.sort((a, b) => b - a);
```

# 14 - Array `flat()` Method

## Description

- The `flat()` method flattens nested arrays into a single array.
- It creates a new array.
- Default depth is `1`.

## Syntax

```js
array.flat(depth)
```

## Example

```js
{
    let arr = [1, 2, [3, 4], [5, 6]];

    let result = arr.flat();

    console.log(
        "14 - Array 'flat()' Method = ",
        result
    );
}
```

---

# Flatten Deep Nested Array

## Example

```js
{
    let arr = [1, [2, [3, [4]]]];

    console.log(arr.flat(3));
}
```

# 20 - Array `some()` Method

## Description

- The `some()` method checks whether at least one element satisfies a condition.

## Syntax

```js
array.some(callback)
```

## Example

```js
{
    let numbers = [2, 4, 7, 8];

    let result = numbers.some(
        num => num % 2 !== 0
    );

    console.log(
        "20 - Array 'some()' Method = ",
        result
    );
}
```

---

# Difference Between `every()` and `some()`

| `every()` | `some()` |
|---|---|
| All elements must satisfy condition | At least one element must satisfy condition |
| Returns `true` only if all pass | Returns `true` if one passes |
