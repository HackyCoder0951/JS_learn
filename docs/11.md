# Chapter 2 - Array Methods for Looping

---

# Introduction

## What are Array Looping Methods?

- Array looping methods are modern ways to iterate over arrays.
- They are cleaner and more readable than traditional loops.
- Commonly used in:
  - React
  - Node.js
  - APIs
  - Data processing
  - Functional programming

---

# Why Use Array Methods Instead of Loops?

| Traditional Loops | Array Methods |
|---|---|
| More code | Cleaner syntax |
| Manual iteration | Automatic iteration |
| Harder to read | Easy to read |
| More error-prone | Safer |

---

# Important Array Looping Methods

| Method | Purpose |
|---|---|
| `forEach()` | Iterate array |
| `map()` | Transform array |
| `filter()` | Filter elements |
| `reduce()` | Reduce to single value |
| `find()` | Find first matching element |
| `findIndex()` | Find index |
| `some()` | Check one condition |
| `every()` | Check all conditions |

---

# Callback Function

## Description

- Most array methods use callback functions.

## Syntax

```js
array.method(function(element, index, array) {

    // code
});
```

---

# Arrow Function Syntax

```js
array.method((element) => {

    // code
});
```

---

# 01 - Array `forEach()` Method

## Description

- The `forEach()` method executes a function for each array element.
- It does not return anything.

## Syntax

```js
array.forEach(callback)
```

---

# Example

```js
{
    let fruits = [
        "Apple",
        "Banana",
        "Mango"
    ];

    fruits.forEach((fruit) => {

        console.log(
            "01 - Array 'forEach()' Method = ",
            fruit
        );
    });
}
```

---

# Access Index in `forEach()`

## Example

```js
{
    let fruits = [
        "Apple",
        "Banana",
        "Mango"
    ];

    fruits.forEach((fruit, index) => {

        console.log(index, fruit);
    });
}
```

---

# Important Note

- `forEach()` cannot:
  - use `break`
  - use `continue`
  - return new array

---

# 02 - Array `map()` Method

## Description

- The `map()` method creates a new array.
- Used for transforming data.

## Syntax

```js
array.map(callback)
```

---

# Example

```js
{
    let numbers = [1, 2, 3, 4];

    let doubled = numbers.map((num) => {

        return num * 2;
    });

    console.log(
        "02 - Array 'map()' Method = ",
        doubled
    );
}
```

---

# Short Arrow Function

## Example

```js
{
    let numbers = [1, 2, 3];

    let result = numbers.map(
        num => num * 10
    );

    console.log(result);
}
```

---

# Real World Example - Product Prices

```js
{
    let prices = [100, 200, 300];

    let gstPrices = prices.map(
        price => price + 18
    );

    console.log(gstPrices);
}
```

---

# Difference Between `forEach()` and `map()`

| `forEach()` | `map()` |
|---|---|
| Returns `undefined` | Returns new array |
| Used for iteration | Used for transformation |

---

# 03 - Array `filter()` Method

## Description

- The `filter()` method creates a new array containing only matching elements.

## Syntax

```js
array.filter(callback)
```

---

# Example

```js
{
    let numbers = [10, 20, 30, 40];

    let result = numbers.filter(
        num => num > 20
    );

    console.log(
        "03 - Array 'filter()' Method = ",
        result
    );
}
```

---

# Real World Example - Active Users

```js
{
    let users = [
        {
            name: "Rahul",
            active: true
        },
        {
            name: "Aman",
            active: false
        }
    ];

    let activeUsers = users.filter(
        user => user.active
    );

    console.log(activeUsers);
}
```

---

# 04 - Array `reduce()` Method

## Description

- The `reduce()` method reduces array values into a single value.

## Syntax

```js
array.reduce(callback, initialValue)
```

---

# Flow of `reduce()`

```text
Accumulator + CurrentValue = Result
```

---

# Example - Sum of Numbers

```js
{
    let numbers = [1, 2, 3, 4];

    let total = numbers.reduce(
        (accumulator, current) => {

            return accumulator + current;

        }, 0
    );

    console.log(
        "04 - Array 'reduce()' Method = ",
        total
    );
}
```

---

# Example - Multiply Numbers

```js
{
    let numbers = [1, 2, 3, 4];

    let result = numbers.reduce(
        (acc, curr) => acc * curr,
        1
    );

    console.log(result);
}
```

---

# Real World Example - Shopping Cart Total

```js
{
    let cart = [
        500,
        1000,
        200
    ];

    let total = cart.reduce(
        (sum, price) => sum + price,
        0
    );

    console.log(total);
}
```

---

# 05 - Array `find()` Method

## Description

- The `find()` method returns the first matching element.

## Syntax

```js
array.find(callback)
```

---

# Example

```js
{
    let users = [
        {
            id: 1,
            name: "Rahul"
        },
        {
            id: 2,
            name: "Jignesh"
        }
    ];

    let user = users.find(
        user => user.id === 2
    );

    console.log(
        "05 - Array 'find()' Method = ",
        user
    );
}
```

---

# 06 - Array `findIndex()` Method

## Description

- The `findIndex()` method returns the index of first matching element.

## Syntax

```js
array.findIndex(callback)
```

---

# Example

```js
{
    let numbers = [10, 20, 30];

    let index = numbers.findIndex(
        num => num === 20
    );

    console.log(
        "06 - Array 'findIndex()' Method = ",
        index
    );
}
```

---

# 07 - Array `some()` Method

## Description

- The `some()` method checks whether at least one element satisfies condition.

## Syntax

```js
array.some(callback)
```

---

# Example

```js
{
    let marks = [45, 80, 90];

    let result = marks.some(
        mark => mark < 50
    );

    console.log(
        "07 - Array 'some()' Method = ",
        result
    );
}
```

---

# 08 - Array `every()` Method

## Description

- The `every()` method checks whether all elements satisfy condition.

## Syntax

```js
array.every(callback)
```

---

# Example

```js
{
    let ages = [18, 20, 25];

    let result = ages.every(
        age => age >= 18
    );

    console.log(
        "08 - Array 'every()' Method = ",
        result
    );
}
```

---

# Difference Between `filter()` and `find()`

| `filter()` | `find()` |
|---|---|
| Returns all matching elements | Returns first matching element |
| Returns array | Returns single value |

---

# Difference Between `some()` and `every()`

| `some()` | `every()` |
|---|---|
| One condition must pass | All conditions must pass |
| Returns quickly on first match | Checks until failure |

---

# Chaining Array Methods

## Description

- Multiple array methods can be chained together.

## Example

```js
{
    let numbers = [1, 2, 3, 4, 5];

    let result = numbers
        .filter(num => num % 2 === 0)
        .map(num => num * 10);

    console.log(result);
}
```
