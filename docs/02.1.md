# JavaScript Interaction Methods (`alert`, `prompt`, `confirm`)

---

# Introduction

JavaScript provides built-in browser functions to interact with users.

The three main interaction methods are:

1. `alert()`
2. `prompt()`
3. `confirm()`

These methods are browser-specific and create popup modal windows.

---

# What is a Modal Window?

A **modal window** is a popup that pauses interaction with the webpage until the user responds.

While a modal window is open:

- The user cannot interact with the rest of the page
- Script execution pauses
- The user must close the popup first

---

# 01 - `alert()`

The `alert()` function displays a message box with an **OK** button.

---

## Syntax

```js
alert(message);
```

---

## Example

```js
alert("Hello");
```

---

## Output

A popup window appears containing:

```txt
Hello
```

with an **OK** button.

---

## Features of `alert()`

- Displays information/messages
- Stops code execution until OK is pressed
- Used for debugging or notifications

---

## Example

```js
alert("Welcome to JavaScript!");
console.log("After alert");
```

The console log executes only after the popup is closed.

---

# 02 - `prompt()`

The `prompt()` function displays:

- A message
- An input field
- OK and Cancel buttons

It is used to collect user input.

---

# Syntax

```js
result = prompt(title, [default]);
```

---

# Parameters

| Parameter | Description |
|---|---|
| `title` | Message shown to the user |
| `default` | Optional default input value |

---

## Note About `[default]`

Square brackets `[]` mean the parameter is optional.

---

# Example

```js
let age = prompt("How old are you?", 18);
```

---

# How It Works

- User enters a value and presses **OK**
- That value is returned as a string
- If user presses **Cancel** or `Esc`, result becomes `null`

---

# Example with Output

```js
let age = prompt("How old are you?", 18);

alert(`You are ${age} years old!`);
```

---

## Possible Outputs

If user enters:

```txt
25
```

Then output:

```txt
You are 25 years old!
```

---

# Cancel Example

```js
let name = prompt("Enter your name");

console.log(name);
```

If user presses Cancel:

```js
null
```

---

# Important Note

`prompt()` always returns:

- `string`
- or `null`

Even numbers are returned as strings.

Example:

```js
let age = prompt("Enter age");

console.log(typeof age);
```

Output:

```js
string
```

---

# 03 - `confirm()`

The `confirm()` function displays:

- A question
- OK button
- Cancel button

It returns a boolean value.

---

# Syntax

```js
result = confirm(question);
```

---

# Return Values

| Button Clicked | Return Value |
|---|---|
| OK | `true` |
| Cancel / Esc | `false` |

---

# Example

```js
let isBoss = confirm("Are you the boss?");
```

---

# Example with Output

```js
alert(isBoss);
```

If user presses:

- OK → `true`
- Cancel → `false`

---

# Complete Example

```js
alert("Hello");

let age = prompt("How old are you?", 100);

alert(`You are ${age} years old!`);

let isBoss = confirm("Are you the boss?");

alert(isBoss);
```

---

# Explanation of Complete Flow

---

## Step 1 - `alert()`

```js
alert("Hello");
```

Shows greeting popup.

---

## Step 2 - `prompt()`

```js
let age = prompt("How old are you?", 100);
```

- Asks user age
- Default value = `100`

---

## Step 3 - Display Input

```js
alert(`You are ${age} years old!`);
```

Displays entered value.

---

## Step 4 - `confirm()`

```js
let isBoss = confirm("Are you the boss?");
```

Asks yes/no question.

---

## Step 5 - Display Boolean Result

```js
alert(isBoss);
```

Displays:

- `true`
- or `false`

---

# Important Features of All Three Methods

| Feature | alert | prompt | confirm |
|---|---|---|---|
| Shows Popup | Yes | Yes | Yes |
| Takes Input | No | Yes | No |
| Returns Value | No | String / null | Boolean |
| Modal Window | Yes | Yes | Yes |

---

# Limitations of These Methods

---

## 1. Browser Controls Appearance

- Popup style depends on browser
- Cannot fully customize design

---

## 2. Position Controlled by Browser

Usually displayed at center of screen.

---

## 3. Blocks User Interaction

User cannot interact with page until popup closes.

---

# Browser vs Node.js

These methods work in browsers only.

---

## Browser

```js
alert("Hello");
```

Works correctly.

---

## Node.js

```js
alert("Hello");
```

Produces error because Node.js has no browser window.

---

## Node.js Alternative

```js
console.log("Hello");
```

---

# Best Practices

✅ Use `alert()` for simple notifications  
✅ Use `prompt()` for small user inputs  
✅ Use `confirm()` for yes/no confirmations  
❌ Avoid excessive popup usage because it affects user experience

---

# Real World Example

```js
let userName = prompt("Enter your name");

let isStudent = confirm("Are you a student?");

alert(`Welcome ${userName}`);
```

---

# Summary

JavaScript provides three built-in interaction methods:

---

## `alert()`

- Displays message
- Only OK button

---

## `prompt()`

- Takes user input
- Returns string or null

---

## `confirm()`

- Asks confirmation question
- Returns true or false

---

# Final Notes

- All popup methods are modal
- They pause script execution
- Mostly used for learning, testing, and simple interactions
- Modern websites usually use custom modal libraries for better UI
