# JavaScript Type Conversion and Operations

---

# Introduction

JavaScript automatically converts values between different data types when needed.  
This process is called **Type Conversion** or **Type Coercion**.

JavaScript supports:

1. Explicit Conversion (manual conversion)
2. Implicit Conversion (automatic conversion)

---

# Explicit Type Conversion

Explicit conversion means manually converting one data type into another.

JavaScript provides built-in conversion functions:

- `Number()`
- `Boolean()`
- `String()`

---

# 01 - Number Conversion

The `Number()` function converts values into numbers.

---

# Syntax

```js
Number(value)
```

---

# Example

```js
let score = "33";

let valueInNumber = Number(score);

console.log(valueInNumber);
console.log(typeof valueInNumber);
```

Output:

```js
33
number
```

---

# Invalid Number Conversion

```js
let score = "jignesh";

let valueInNumber = Number(score);

console.log(valueInNumber);
```

Output:

```js
NaN
```

---

# What is `NaN`?

`NaN` means:

```txt
Not a Number
```

It represents invalid numeric conversion.

---

# Common Number Conversions

| Value | Result |
|---|---|
| `"33"` | `33` |
| `"33abc"` | `NaN` |
| `true` | `1` |
| `false` | `0` |
| `null` | `0` |
| `undefined` | `NaN` |

---

# Example

```js
console.log(Number(true));
console.log(Number(false));
```

Output:

```js
1
0
```

---

# 02 - Boolean Conversion

The `Boolean()` function converts values into `true` or `false`.

---

# Syntax

```js
Boolean(value)
```

---

# Example

```js
let isLoggedIn = "jignesh";

let booleanIsLoggedIn = Boolean(isLoggedIn);

console.log(booleanIsLoggedIn);
```

Output:

```js
true
```

---

# Common Boolean Conversions

| Value | Result |
|---|---|
| `1` | `true` |
| `0` | `false` |
| `""` | `false` |
| `"jignesh"` | `true` |
| `null` | `false` |
| `undefined` | `false` |

---

# Example

```js
console.log(Boolean(""));
console.log(Boolean("Hello"));
```

Output:

```js
false
true
```

---

# String Concatenation

Strings combine using the `+` operator.

---

# Example

```js
let str1 = "hello";
let str2 = " Jignesh";

let str3 = str1 + str2;

console.log(str3);
```

Output:

```js
hello Jignesh
```

---

# Important String + Number Behavior

JavaScript converts numbers into strings when using `+` with strings.

---

# Example 1

```js
console.log("1" + 2);
```

Output:

```js
12
```

---

# Example 2

```js
console.log(1 + "2");
```

Output:

```js
12
```

---

# Example 3

```js
console.log("1" + 2 + 2);
```

Output:

```js
122
```

Explanation:

- `"1" + 2` → `"12"`
- `"12" + 2` → `"122"`

---

# Example 4

```js
console.log(1 + 2 + "2");
```

Output:

```js
32
```

Explanation:

- `1 + 2` → `3`
- `3 + "2"` → `"32"`


# JavaScript Type Coercion Examples

---

# Example 1

```js
"" + 1 + 0
```

Output:

```js
"10"
```

Explanation:

- `"" + 1` → `"1"`
- `"1" + 0` → `"10"`

---

# Example 2

```js
"" - 1 + 0
```

Output:

```js
-1
```

Explanation:

- Empty string becomes `0`
- `0 - 1` → `-1`

---

# Example 3

```js
true + false
```

Output:

```js
1
```

Explanation:

- `true` → `1`
- `false` → `0`

---

# Example 4

```js
6 / "3"
```

Output:

```js
2
```

String `"3"` converts into number `3`.

---

# Example 5

```js
"2" * "3"
```

Output:

```js
6
```

Both strings convert into numbers.

---

# Example 6

```js
4 + 5 + "px"
```

Output:

```js
"9px"
```

---

# Example 7

```js
"$" + 4 + 5
```

Output:

```js
"$45"
```

---

# Example 8

```js
"4" - 2
```

Output:

```js
2
```

Subtraction converts string into number.

---

# Example 9

```js
"4px" - 2
```

Output:

```js
NaN
```

`"4px"` cannot convert into valid number.

---

# Example 10

```js
"  -9  " + 5
```

Output:

```js
"  -9  5"
```

Because `+` performs string concatenation.

---

# Example 11

```js
"  -9  " - 5
```

Output:

```js
-14
```

Subtraction converts string into number.

---

# Example 12

```js
null + 1
```

Output:

```js
1
```

`null` becomes:

```js
0
```

---

# Example 13

```js
undefined + 1
```

Output:

```js
NaN
```

---

# Example 14

```js
" \t \n" - 2
```

Output:

```js
-2
```

Whitespace converts into `0`.

---

# Numeric Conversion Rules

| Value | Converts To |
|---|---|
| `undefined` | `NaN` |
| `null` | `0` |
| `true` | `1` |
| `false` | `0` |
| Empty String | `0` |
| Invalid String | `NaN` |

---

# Whitespace Handling

When converting strings into numbers:

- Spaces
- Tabs (`\t`)
- Newlines (`\n`)

are automatically trimmed.

# JavaScript Conversion Flow

---

# String Conversion

```js
String(value)
```

Converts any value into string.

---

# Number Conversion

```js
Number(value)
```

Converts values into numbers.

# Boolean Conversion

```js
Boolean(value)
```

Converts values into true/false.

---

# Important Notes

✅ `+` with string performs concatenation  
✅ Mathematical operators convert values into numbers  
✅ `NaN` means invalid numeric conversion  
✅ JavaScript automatically performs implicit conversions  
✅ Empty strings become `0` in numeric conversion
